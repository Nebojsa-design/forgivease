Data Deletion

Delete ForgivEase Data

The current MVP keeps practice history on the device and gives users local deletion controls.

Current MVP scope: ForgivEase has no user accounts and no ForgivEase backend profile in the current MVP.

Delete One Saved Process

  1. Open Processes.
  2. Open the saved process.
  3. Tap Delete this process.
  4. Confirm local removal.

Remove One Voice Note

  1. Open Processes.
  2. Open the saved process.
  3. Tap Remove voice note.
  4. Confirm the voice-note area shows the empty state.

Delete All Local Data

  1. Open Settings.
  2. Tap Delete local data.
  3. Confirm deletion.
  4. Open Processes and confirm the empty state.

Clear A Draft

  1. Open Home.
  2. Use Clear draft when draft recovery appears.
  3. Confirm the draft card disappears.
  4. Start a new practice if needed.

What Deletion Does

The current MVP deletes local practice records from device storage. Native builds also attempt to delete local native voice-note audio files when voice notes, saved processes, or all local data are removed.

User-managed copies can remain outside the app after export, screenshots, device backup, file sharing, manual copying, or operating system behavior.

Account Deletion

The current MVP has no user accounts and no ForgivEase backend profile. Account deletion will be added before account sync, subscriptions, AI cloud features, public groups, or backend storage.
